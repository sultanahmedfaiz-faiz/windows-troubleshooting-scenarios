# Scenario 2: Network Printer Not Printing

## Issue Description
User reported that documents are stuck in the print queue and not printing.

## Environment
- Windows 10 Pro
- Network Printer
- Connected via print server

## Troubleshooting Steps
1. Checked printer power and network connectivity
2. Restarted Print Spooler service
3. Cleared stuck print jobs
4. Verified correct printer driver installation
5. Set printer as default

## Root Cause
Print Spooler service was stuck due to a corrupted print job.

## Resolution
Restarted Print Spooler service and cleared the print queue.

## Outcome
Printing resumed successfully and user confirmed issue resolved.
