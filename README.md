# Windows 10/11 Troubleshooting Scenarios

## Project Overview
This project contains real-world Windows 10 and Windows 11 troubleshooting scenarios
commonly handled by IT Support and Service Desk engineers.

The scenarios are based on typical user-reported issues in enterprise environments
and include structured troubleshooting steps and resolutions.

## Skills Demonstrated
- Windows 10 & Windows 11 troubleshooting
- IT support ticket handling
- Root cause analysis
- Documentation and communication
- Basic system administration

## Tools & Technologies
- Windows 10 / Windows 11
- Task Manager
- Event Viewer
- Command Prompt / PowerShell
- Device Manager

## Scenario List
- Slow system boot issue
- Printer not working
- Outlook not syncing emails
- Wi-Fi connectivity issues
- Disk space running low

## How to Use This Repository
Each scenario is documented in a separate markdown file with:
- Issue description
- Environment details
- Troubleshooting steps
- Resolution
- Outcome
