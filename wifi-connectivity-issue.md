# Scenario 3: Wi-Fi Connectivity Issue

## Issue Description
User reported frequent disconnections from corporate Wi-Fi.

## Environment
- Windows 11
- Laptop
- Corporate Wi-Fi network

## Troubleshooting Steps
1. Checked Wi-Fi signal strength
2. Updated network adapter drivers
3. Ran network troubleshooting tool
4. Reset network settings
5. Tested connectivity using ping

## Root Cause
Outdated network adapter driver.

## Resolution
Updated Wi-Fi adapter driver and reset network settings.

## Outcome
Stable Wi-Fi connection restored.
